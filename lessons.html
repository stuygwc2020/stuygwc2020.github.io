CS @ Stuy Panel

February 7th, 2020

We kicked off the first meeting of second semester by introducing the computer science department at Stuyvesant. Sophomores are required to take an introductory CS course, in the form of either Intro CS (one semester) or Annual CS (two semesters). In Intro CS, students learn about Racket and NetLogo; Annual students learn Python and HTML. We stressed the importance of taking Annual CS if you have even a slight interest in computer science. Then, the presidents demoed Pac-Man from the NetLogo Model Library and further elaborated on the visual differences between NetLogo and Racket. We then dived into AP CS, a course open to juniors and seniors. The APCS curriculum is taught in Java and focuses on object-oriented programming and utilizing problem-solving skills. We learned that the entire CollegeBoard APCS curriculum is taught in the fall semester, while data structures and algorithms are taught in the spring semester. Annual CS prepares students better for APCS; however, it is still possible for Intro students to take APCS. Afterwards, we demoed Tammy and Jenny’s APCS Project, a remake of the game Plants vs Zombies. Afterwards, we talked about several post-APCS electives: Software Development, System Level Programming, Computer Graphics, Artificial Intelligence, and a new elective coming next year, Cybersecurity. Jackie demoed the projects she made in her Software Development class, including a TriviaCrack project that quizzed users on a wide range of topics and a casino game, which allowed the user to play Blackjack and scratch lottery tickets. She said that Software Development strengthened her project management and teamwork skills as well as her understanding of web-based applications and full-stack development. Afterwards, Tammy talked about her experience in System Level Programming. Tammy clarified System Level Programming was about C (a low-level programming language), development in the Linux environment, the main components of a computer, and networking (server and client programs). Next, we talked about Computer Graphics, an animation course that allows you to understand and use motion description, lighting and shading, animation, ray tracing, and graphics algorithms. Finally, we talked about Artificial Intelligence, a course that discusses the ethics of AI as well as topics such as neural networks. We wrapped up the lesson by talking about CS programs outside of Stuy. One well-known program is called GWC Summer Immersion, a free, 7-week computer course that teaches a wide variety of coding languages through lessons, guest speakers, and trips. Additionally, once you complete the course, you gain access to the GWC alumni network. Next, we learned about a project-based program hosted by Cooper Union. The first 2 weeks of this 6-week program focused on web development in order to create a fully functional web application which tackles a UN Sustainable Developmental Goal. Finally, we learned about the BNY Mellon internship, which immerses participants in the corporate world of computer science and teaches networking skills. We then briefly talked about several other programs such as Kode with Klossy and the Brown Scholars CS Program. At the end of the meeting, the board reminded everyone that Mr. Blumm’s Opportunities Bulletin is a great resource for finding summer programs.



Minecraft Redstone

January 10th, 2020

Kicking off our first meeting of 2020, we learned about Minecraft Redstoning. George first explained Redstone was electricity in minecraft. We then went on a website that demonstrated the uses of Redstone in minecraft from a bird’s eye view. Afterwards, we learned how redstone torches supply electricity, levers allow the player to control the electricity, and buttons allow the player to release short bursts of electricity. Afterwards, we learned how to delete blocks in the simulator. We figured out the maximum length of a signal is 15 blocks, so we would need repeaters to strengthen the signal. We then used our knowledge to build simple contraptions.

Intro to Javascript

In today's lesson, we learned more about styling in JavaScript. We refreshed the commands we learned in terminal from the previous lesson. CSS Continued

November 15th, 2019

Today, we continued applying CSS to our HTML websites. We started off by learning about the three methods to add colors in CSS: name, RGB values, and hex values. We also learned how to add comments into our stylesheets. We focused mostly on using hex values and to solidify our understanding, we played a game called Hex Invaders. We then learned about organization in HTML, using div tags to separate content into different sections. We also learned to add class and ID attributes to our HTML tags, which allows for more versatile styling. We ended the meeting with CSS practice from CodeAcademy. Additionally, we announced our first guest speaker, Isabel Abonitalla!



Introduction to CSS

November 8th, 2019

Today, we learned how to style our HTML pages using CSS (Cascading Style Sheets). We used Inspect Element to examine the styling of other websites. Afterwards, we learned the syntax of CSS and the three methods of implementing it. We learned about inline styling, which is directly inside HTML tags, internal stylesheet, which is written inside a style tag, and external stylesheet, which is situated in a CSS file and linked to the HTML code. We then learned about other properties we can modify in CSS, such as font-size, color, background-color, border-radius and width. We ended the meeting with a fun game of Kahoot!
